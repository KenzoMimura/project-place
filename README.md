# Project Place

A quiet place for personal writing, self-reflection, and thoughts still finding their words.

## Sobre o projeto

Este é meu primeiro projeto autoral, criado a partir da ideia de construir um espaço minimalista, privado e humano para escrita pessoal e autoconhecimento.

O objetivo não é criar mais uma rede social, app de notas ou ferramenta de produtividade. A proposta é oferecer um lugar silencioso onde a pessoa possa escrever com calma, sem distrações, sem comparação e sem pressão.
